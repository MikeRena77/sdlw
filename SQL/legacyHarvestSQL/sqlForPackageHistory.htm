Hi Felipe,

if I understand you, you want to create a package history report. For this I give you the SQL-Command to get the packagename, action, State and execution time of the package.

SELECT HARPACKAGE.PACKAGENAME,
HARPKGHISTORY.ACTION,
HARPKGHISTORY.STATENAME,
HARPKGHISTORY.EXECDTIME
FROM HARPACKAGE HARPACKAGE,
HARPKGHISTORY HARPKGHISTORY
WHERE HARPACKAGE.PACKAGEOBJID = HARPKGHISTORY.PACKAGEOBJID
ORDER BY HARPACKAGE.PACKAGENAME, HARPKGHISTORY.EXECDTIME

Hope that helps
